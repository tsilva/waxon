# Experiments

* [Learn widget repair and tutor latency](learn-widget-repair-latency-2026-06-23.md) - Native-browser Learn evaluation found partial hidden widget comments could swallow fallback widgets, and recorded the tutor latency improvement after repair.
* [Learn prompt cache on OpenRouter Gemini](learn-prompt-cache-openrouter-2026-06-24.md) - Real Learn turns found Gemini 3.1 Flash Lite returned zero cache writes, while Gemini 2.5 Flash produced verified cache writes/reads with immutable tutor instructions cached.
* [Learn answer evaluator cache on OpenRouter Mercury](learn-answer-evaluator-cache-2026-06-24.md) - Real Learn turns found explicit cacheable text blocks partially improve Mercury evaluator cache reads, while padded Gemini evaluator prompts cached more tokens but were slower.
